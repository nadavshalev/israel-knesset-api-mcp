# ----------------------------------------
# Essentials
## activate the virtual environment
```bash
source .venv/bin/activate
```

## run Hebrew support terminal
```bash
konsole &
```

## run sqlite browser
```bash
sqlitebrowser data.sqlite
```

## run all tests
```bash
pytest -v
```

# MCP
## run the MCP server
```bash
python mcp_server.py 
```

## run MCP Inspection Tool
```bash
npx @modelcontextprotocol/inspector
```
